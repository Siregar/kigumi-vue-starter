# Kigumi Vue Starter

Vue starter project powered by [Kigumi](https://kigumi.style). Use it to quickly start a new Vue project with Kigumi.

## Prerequisites

- Node.js 20+

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── components/
│   ├── examples/        # Example screens (LoginExample, ...)
│   └── ui/              # Kigumi-managed Vue wrappers (Button, Input, ...)
├── lib/kigumi.ts        # Web Awesome setup and theme registration
├── styles/
│   ├── layers.css       # CSS cascade layer definitions
│   └── theme.css        # Custom theme overrides
└── App.vue              # Application entry
```

Component wrappers in `components/ui/` are generated by the Kigumi CLI. They provide typed Vue components around Web Awesome custom elements with `v-model` support, event binding, and slot forwarding.

## Adding Components

```bash
npx kigumi add <component>
```

## AI Skills

Kigumi provides a set of AI skills to help you with your project.
To install the skills, run the following command:

```
npx skills add https://kigumi.style/skills/kigumi
```

## Scripts

| Command           | Description                         |
| ----------------- | ----------------------------------- |
| `npm run dev`     | Start Vite dev server               |
| `npm run build`   | Type-check and build for production |
| `npm run lint`    | Run ESLint                          |
| `npm run preview` | Preview production build            |
